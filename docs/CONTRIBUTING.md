# Contribuir

## Requisitos
- Flutter 3.x
- Linux

## Flujo recomendado
1. `flutter pub get`
2. `flutter analyze`
3. `flutter test`

## Estilo
- Mantener UI consistente con NiceOSTheme.
- Evitar dependencias innecesarias.

## Pull Requests
- Describir cambios claramente.
- Adjuntar capturas si hay cambios visuales.
